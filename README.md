🚀 Proyecto: Gestión de Empleados & Departamentos (Prueba Técnica)
Este proyecto es una aplicación Full-Stack desarrollada para gestionar la estructura organizacional de una empresa, permitiendo administrar empleados y sus respectivos departamentos mediante una interfaz intuitiva y profesional.

🛠️ Stack Tecnológico
Frontend: Angular 17+ (Componentes Standalone).

Backend: Node.js con Express (Proyecto bit-back).

Base de Datos: MongoDB.

Diseño UI/UX: Basado en la regla 60-30-10 para una ergonomía visual superior.

📋 Requerimientos Implementados
Operaciones CRUD: Interfaz completa para agregar, visualizar y eliminar empleados y departamentos.

Arquitectura Modular: Separación clara entre componentes de lógica (servicios) y componentes visuales.

Vista Maestra (Dashboard): Una ventana dinámica que organiza a los usuarios según su departamento asignado.

Estilos Globales: Implementación de variables CSS para facilitar el mantenimiento y la escalabilidad del diseño.

🚀 Instalación y Uso
Backend:

Navegar a la carpeta bit-back.

Configurar el archivo .env con la URI de MongoDB.

Ejecutar npm start.

Frontend:

Instalar dependencias: npm install.

Levantar el servidor de desarrollo: ng serve.

Abrir en el navegador: http://localhost:4200/.
http://localhost:4200/dashboard - para ver empleados y departamentos
